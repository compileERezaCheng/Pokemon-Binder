Google Sheets Sync Settings

Automatically synchronize your card binder metadata rows to a Google Spreadsheet.

  1. Open the Google Cloud Console and create a new project.
  2. Enable the Google Sheets API and Google Drive API inside your project.
  3. Go to Credentials → Click + CREATE CREDENTIALS → Select Service Account.
  4. Navigate to the created Service Account's Keys tab, click Add Key → Create new key → select JSON.
  5. Download the JSON credentials file, rename it exactly to credentials.json, and place it in the same directory as your Python files.
  6. Create a Google Spreadsheet (e.g. named "Pokemon Binder") in your Google Drive.
  7. Click Share in your Google Sheet and add the Service Account's email address (found inside your JSON key under client_email) with Editor permission.
  8. Ensure the Spreadsheet Name matches your Google Sheets Sync setting exactly!
